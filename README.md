# reduxBookApp

###Getting Started###

There are two methods for getting started with this repo.

####Familiar with Git?#####
Checkout this repo, install depdencies, then start the gulp process with the following:

```
	> git clone git@github.com:khalilm1906/reduxBookApp.git
	> cd reduxBookApp
	> npm install
	> npm start
```

####Unfamiliar with Git?####
```
    > Download zip file
    > Extract zip file
    > cd reduxBookApp
	> npm install
	> npm start
```
